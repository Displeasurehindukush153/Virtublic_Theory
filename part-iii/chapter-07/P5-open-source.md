## Statement

All system code is open for verification by any citizen.

## Formal notation

```
∀ module ∈ System: source_code = publicly_verifiable
```

## Detailed description

This is not merely transparency — it is a condition of legitimacy. A citizen who cannot verify the code is forced to trust developers. This reproduces the problem of T12.

Open code + formal verification = public verifiability of constitutional invariants.
