## Statement

A guaranteed and unconditional exit from the system — without penalties or algorithmic pressure.

## Formal notation

```
∀ citizen c: ∃ Exit(c) → ¬Data(c) ∧ ¬Pressure(c)
```

## Detailed description

Any citizen may exit the system completely: data is deleted, participation ends, no consequences for civil rights outside the system.

This is the constitutional response to platform retention mechanisms. Exit must be _real_, not formal.
