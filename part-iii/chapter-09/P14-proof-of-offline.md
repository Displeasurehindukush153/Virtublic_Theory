## Statement

Citizens prove offline periods via zk-proof and receive a contribution bonus.

## Formal notation

```
zk-proof(offline_period) → cognitive_health_bonus(VIC⊥)
```

## Detailed description

zk-proof: "I did not use digital devices for N hours" — without surveillance, without transmitting data about _what_ you did offline. The system knows only the fact of the offline period, not its content.

This is the direct and only constitutional response to T6 (cognitive disarmament). Offline time becomes not a loss of participation, but a source of it.
