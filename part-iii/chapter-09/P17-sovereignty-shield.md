## Statement

Any state request for citizen data is a public transaction on the blockchain.

## Formal notation

```
∀ state_request → on-chain_transaction; Mandate: 75% EQU⊥
```

## Detailed description

The state cannot request citizen data secretly: every request automatically creates an on-chain transaction visible to all. The mandate for a request requires 75% EQU⊥ + CivicJuryEngine verification.

Violation automatically activates P18. This is the direct response to the state as buyer of predictions.
