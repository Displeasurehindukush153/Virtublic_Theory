## Statement

Method of elimination: everything except a constitutional institution fails.

## Medium description

Self-regulation excluded (T3). State regulation excluded via conflict of interest. Individual action excluded (T5–T6). Informal collective action excluded (L8). One option remains.

## Formal notation

```
¬self_regulation ∧ ¬state ∧ ¬individual ∧ ¬informal_collective → constitutional_institution
```

## Detailed explanation

This is not inference by analogy — it is a strict proof by elimination. Each alternative is sequentially excluded in previous theorems:

- **T3**: internal reform is impossible without changing the objective function
- **ΣA17**: the state itself is a buyer of predictions
- **T5**: individual resistance is neutralized
- **L8**: critique is monetized and absorbed

The only object that passes the filter is a **constitutional institution**. This is the direct transition to Parts II and III.
