## Statement

Without external constraint the system destroys the distinction between information and manipulation.

## Medium description

The loop (Capture → Predict → Shape → Capture) leads in the long run to an information space entirely managed by predictive models.

## Formal notation

```
∀t > T_critical: distinction(information, manipulation) → 0
```

## Detailed explanation

LLM models optimized for engagement reproduce clickbait as a _structural necessity_. This is not a deviation — it is the optimal solution to the engagement maximization problem. AI amplification makes this process orders of magnitude faster.

The point of no return is not a catastrophe, but a quiet normalization: when people stop noticing manipulation because it has become background noise.
