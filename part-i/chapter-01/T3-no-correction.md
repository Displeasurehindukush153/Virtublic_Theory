## Statement

The system contains no mechanism to reduce the intensity of attention capture.

## Medium description

As long as the objective function — maximizing engagement — remains unchanged, any internal reform merely changes the form of exploitation, not its substance.

## Formal notation

```
¬∃ internal_reform: ¬optimize(engagement)
```

## Detailed explanation

When a platform introduces "screen time" or a "take a break" button — that is not a system correction. It is a new engagement mechanism: the user who receives a notification about 3 hours on TikTok opens TikTok to read it.

This is not developer cynicism — it is a structural constraint. Shareholders demand growth. Growth = engagement. Engagement requires optimization. The circle is closed.
