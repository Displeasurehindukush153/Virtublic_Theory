# The Attention Substrate

_How the capture machine works — at the physics level of the system_

## Level of description

This chapter describes a mechanism. Not malicious intent, not specific individuals — but a structure that operates independently of the intentions of its participants.

## Six Axioms of the Substrate

### A1 — Finitude
Waking time is absolutely limited. ~16 hours per day. It cannot be purchased, it cannot be produced additionally. This limit makes attention valuable — and makes its exploitation structural, not incidental.

### A2 — Non-delegability
Attention cannot be transferred, but it can be alienated. You cannot delegate the act of perception. But the result of your perception — behavioral data — is converted into platform capital without an equivalent exchange.

### A3 — Quantization
Every interaction is irreversible. Attention-token τ = (subject, platform, time) — the elementary unit of alienation. Each τ is irreversible: data about your viewing does not disappear when you close the app.

### A4 — Crystallization
Live attention turns into dead data — permanently. The transition is unidirectional: Subject_time → Capital_data. There is no way back.

### A5 — Aggregation
The value of data grows nonlinearly with scale. Upon reaching critical mass M_crit, the predictive value of data begins to grow disproportionately.

### A6 — Self-amplification
The cycle is closed and has no saturation point.

::formula
Attention → Data → Model → Utility → More attention
∂²C/∂(Σ Attentionᵢ)² ≥ 0  // nonlinear growth
::
