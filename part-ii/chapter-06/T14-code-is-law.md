## Statement

Algorithmic power without constitutional protection of subjecthood is the same arbitrariness, only faceless.

## Medium description

"Code is law" means: rules execute automatically without discretion. This is beneficial for predictability. It is harmful when the code is optimized for efficiency rather than subjecthood.

## Formal notation

```
Optimize(efficiency) ∧ ¬Protect(subjecthood) → Exploitation
```

## Detailed explanation

A smart contract does not know context. If the terms state "seize collateral after 1 day of default" — it will seize it, even if the person was in hospital. No appeal. No discretion.

NA0 (subjecthood as a protected good) must be encoded _before_ any other rules — as an axiom, not as an exception.
