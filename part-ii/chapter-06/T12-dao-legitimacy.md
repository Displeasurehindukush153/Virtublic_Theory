## Statement

The DAO says: "this is legitimate because the code says so". But who wrote the code?

## Medium description

DAO legitimacy is grounded in the code, the code is written by developers and early participants — a circle. This is not a democratic mandate.

## Formal notation

```
Legitimacy(DAO) = Code_says_so ≠ Democratic_mandate
```

## Detailed explanation

This is not a theoretical problem — it is practical. In 2016, The DAO was exploited for $60M. The majority of Ethereum holders voted for a hard fork. The minority split off as Ethereum Classic.

This demonstrated: in a crisis, a DAO always falls back on politics. The question is not whether politics will appear — but whether it will be constitutionally formalized.
