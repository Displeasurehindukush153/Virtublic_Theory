# Blockchain Anthropology

_Five positions — none is a genuine political subject_

## Who participates in a blockchain system?

### Holders
Token holders. Reduced to capital. Influence proportional to stake. 1 token = 1 vote.

### Stakers
Compound interest without creating new value. Reproduces T1 at the capital level.

### Users
Use the infrastructure without participating in governance. External to power.

### Developers
Real power through code, but without a democratic mandate. → T12.

### Critics
A structurally new position: extract capital from diagnosis without an institutional alternative. Absorption → T16.

## Key observation

The Critics position is especially instructive: it emerges precisely when systemic diagnosis becomes profitable. But diagnosis without an alternative is absorption, not resistance.
